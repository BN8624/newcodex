## 2026-06-27 공개 확인 후보 수정

- 이전 빌드는 MVP와 확인용 골격에 가까워 사용자에게 공개 확인 후보로 내밀기에는 부족했다.

- `C:\Users\USER\unknown`은 참고 기준으로만 비교했다. 실제 반영한 점은 프로젝트 이름, 게임 UI, 데이터 이름, 문서, Web 제목이 모두 한국어로 보여야 한다는 것이다.
